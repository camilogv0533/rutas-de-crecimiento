# Positioning — Rutas de Crecimiento

## One-liner
El directorio curado de viajes que desarrollan habilidades, no vacaciones disfrazadas.

## Categoría
Nuevo tipo de turismo: **turismo para el desarrollo de habilidades** (skills tourism / immersive learning travel).

## Diferencial vs competencia
| Plataforma | Qué hace | Por qué no es lo mismo |
|---|---|---|
| BookRetreats | Listado masivo wellness/yoga | No filtra por habilidad, no curado |
| RetreatFinder | Buscador genérico | Mucho ruido, mismo problema |
| BlackTomato | Viajes a medida lujo | $$$, no orientado a desarrollar habilidad concreta |
| TheGlobalRetreatCo | Curado bonito wellness | Sí curado pero no enfocado en habilidad |
| Health.travel | Wellness corporativo | Wellness, no skill-driven |
| Airbnb Experiences | Local experiences | No es viaje inmersivo de aprendizaje |
| EdX / Coursera | Aprendizaje | No es viaje |

Hueco que ocupamos: **intersección viaje × aprendizaje × habilidad concreta**.

## Promesa
"Te decimos qué habilidad desarrollarás, dónde, con quién, y por qué ese lugar es el correcto para esa habilidad."

## Pillars de SEO (las 5 verticales iniciales)
1. **Liderazgo** — keyword "retiro liderazgo", "leadership retreat".
2. **Creatividad** — "retiros para creadores", "creative retreat".
3. **Propósito / claridad profesional** — "retiro propósito profesional", "career retreat".
4. **Resiliencia / burnout** — "retiro burnout ejecutivo", "executive burnout retreat".
5. **Comunicación / oratoria** — "retiro storytelling", "speaking retreat".

## Lo que NO somos (repetir para alineación interna)
- No agencia de viajes.
- No buscador de retiros espirituales genérico.
- No Airbnb / Booking / Tripadvisor.
- No revista de wellness.

## Visión 3 años
Plataforma de L&D online + buscador de experiencias inmersivas. Ingresos por: vendor promotion, sello de auditoría, asesoría HR, marketing afiliados, productos asociados. No por publicidad.
